# Wood Floor Calculator

Calculate square footage of wood flooring from pallet and bundle data.

## Live Demo

🔗 [procodcj.github.io/wood-floor-calculator](https://procodcj.github.io/wood-floor-calculator)

## What it does

Enter a few measurements and instantly get the square footage breakdown — per board, per bundle, and per full pallet.

## Inputs

| Field | Description |
|---|---|
| Bundle Length | Length of the boards in a bundle (ft or in) |
| Board Width | Width of each board (ft or in) |
| Boards per Bundle | How many boards are in one bundle |
| Bundles per Pallet | How many bundles make up the pallet |

## Output

- **Total sq ft — Pallet**
- **Sq ft per Bundle**
- **Sq ft per Board**

## Features

- Accepts whole numbers, decimals, and fractions (e.g. `12 3/4`, `5 1/2`)
- Supports inches and feet for both length and width
- Input validation with clear error messages
